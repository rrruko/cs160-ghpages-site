Chapter 11

Briefly describe the four gates explained in Chapter 11. Describe the behavior of each gate based on the input values to the gate.

OR gate: output 1 if any inputs are 1, otherwise 0.
AND gate: output 1 if all inputs are 1, otherwise 0.
NOR gate: output 1 if no inputs are 1, otherwise 0.
NAND gate: output 1 unless all inputs are 1.

Chapter 12

A half adder is built from how many "sub components" and has how many inputs and outputs?

Two components: an AND gate and an XOR gate (which is itself probably composed of an OR, a NAND, and an AND gate).

Two inputs (one for each bit) and two outputs, for the sum and the carry.

A full adder is built from how many "sub components" and has how many inputs and outputs?

Three components: two half adders and an OR gate.

Three inputs and two outputs for sum and carry.

How many total inputs and outputs are there for an 8-bit adding circuit?

Seventeen inputs for each bit of the two 8-bit inputs, plus an extra because we're using full adders. Nine outputs for each bit of the output. The ninth output might be considered extraneous too since computers add with overflow, writing the result of an 8-bit addition to yet another 8-bit number, even if the result is incorrect*. However, some (most? all?) CPUs keep track of whether the last arithmetic operation resulted in a carry, making the carry out bit useful to machine code.

*The statement 255 + 1 = 0 isn't actually wrong if you're talking about addition modulo 256.

How many total transistors are needed for the ripple version of the 8-bit adder?

144.
