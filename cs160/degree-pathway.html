OSU Cascades's undergrad CS program is geared toward web and mobile software development. Both Corvallis and Cascades require students to apply to the Engineering Professional Program to be eligible to register for higher-level courses, which involves passing a certain set of more general courses.

Grad Program

OSU Cascades campus does not offer a graduate program in CS, so graduate courses have to be taken at Corvallis campus.

The graduate CS program offers an M.S., M.Eng., and Ph.D.
